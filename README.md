# Ymlmailer

tiny mailer program using YAML-based mail data file.

## Installation
Create gem file.

    $ rake build

install it yourself as:

    $ gem install pkg/ymlmailer-x.xx.gem


## Usage

    $ ymlmailer -f <mail data file>
